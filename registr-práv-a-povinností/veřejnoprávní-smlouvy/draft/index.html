Abstrakt

Tento dokument je otevřenou formální normou ve smyslu § 3 odst. 9 zákona č. 106/1999 Sb., o svobodném přístupu k informacím, pro zveřejňování informací o veřejnoprávních smlouvách k přenesení působnosti z Registru práv a povinností ve smyslu § 51 odst. 1 zákona č. 111/2009 Sb. o základních registrech v podobě otevřených dat. Norma specifikuje způsob publikace na stupni otevřenosti 3* a 5* ve formátu JSON-LD.

JSON struktura

Přehled JSON struktury

Datová sada je tvořena seznamem instancí typu Veřejnoprávní smlouva k předání působnosti uvedených v poli položky.
  • Instance typu Veřejnoprávní smlouva k předání působnosti má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^smlouva$
    • id : Odkaz dle vzoru ^smlouva/.+$
    • identifikátor : Řetězec
    • uzavření : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
    • ukončení : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
    • na-dobu-neurčitou : Boolean
    • rozhodnutí-o-předání : instance typu Rozhodnutí o založení veřejnoprávní smlouvy
    • rozhodnutí-o-převzetí : instance typu Rozhodnutí o založení veřejnoprávní smlouvy
    • rozhodnutí-o-schválení : instance typu Rozhodnutí o schválení předání působnosti
    • schvaluje : Odkaz dle vzoru ^orgán-veřejné-moci/[0-9]+-[0-9]+$
    • převádí : Odkaz dle vzoru ^orgán-veřejné-moci/[0-9]+-[0-9]+$
    • přebírá : Odkaz dle vzoru ^orgán-veřejné-moci/[0-9]+-[0-9]+$
  • Instance typu Rozhodnutí o schválení předání působnosti má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^rozhodnutí-schválení$
    • id : Odkaz dle vzoru ^rozhodnutí/.+$
    • číslo-jednací : Řetězec
    • datum : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
    • typ-rozhodnutí-schválení : Odkaz dle vzoru ^typ-rozhodnutí-schválení/[0-9]+$
  • Instance typu Rozhodnutí o založení veřejnoprávní smlouvy má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^rozhodnutí-založení$
    • id : Odkaz dle vzoru ^rozhodnutí/.+$
    • číslo-jednací : Řetězec
    • datum : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
