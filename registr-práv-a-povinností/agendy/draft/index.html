Abstrakt

Tento dokument je otevřenou formální normou ve smyslu § 3 odst. 9 zákona č. 106/1999 Sb., o svobodném přístupu k informacím, pro zveřejňování informací o agendách z Registru práv a povinností ve smyslu § 51 odst. 1 zákona č. 111/2009 Sb. o základních registrech v podobě otevřených dat. Norma specifikuje způsob publikace na stupni otevřenosti 3* a 5* ve formátu JSON-LD.

Přehled JSON struktury

Datová sada je tvořena seznamem instancí typu Agenda uvedených v poli položky.
  • Instance typu Agenda má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^agenda$
    • id : Řetězec dle vzoru ^agenda/A[0-9]+$
    • název : Název agendy v různých jazycích
    • kód : Řetězec dle vzoru ^A[0-9]+$
    • ustanovení : seznam instancí typu Ustanovení právního předpisu
    • činnosti : seznam instancí typu Činnost
    • vykonávající-ovm : seznam odkazů
    • vykonávající-kategorie-ovm : seznam odkazů
    • vykonávající-spuú : seznam odkazů
    • vykonávající-kategorie-spuú : seznam odkazů
    • ohlašovatel : Řetězec dle vzoru ^orgán-veřejné-moci/[0-9]{8}-[0-9]+$
    • vznik : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
    • zánik : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
    • platnost-od : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
    • platnost-do : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
    • poslední-změna : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
  • Instance typu Ustanovení právního předpisu má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^ustanovení-právního-předpisu$
    • označení : Řetězec
  • Instance typu Činnost má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^činnost$
    • id : Řetězec dle vzoru ^činnost/A[0-9]+/CR[0-9]+$
    • kód-činnosti : Řetězec dle vzoru ^CR[0-9]+$
    • název-činnosti : Název činnosti v různých jazycích
    • popis-činnosti : Popis činnosti v různých jazycích
    • typ-činnosti : Řetězec dle vzoru ^typ-činosti/[0-9]+$
    • platnost-činnosti-od : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$
    • platnost-činnosti-do : Řetězec dle vzoru ^[0-9]{4}-[0-9]{2}-[0-9]{2}$

Příklady

Základní identifikační údaje o agendě

Následující příklad demonstruje reprezentaci základních identifikačních údajů agendy - id, identifikátor, název a kód. Rozdíl mezi id a kódem není v kontextu Registru práv a povinností žádný. Kód je prostý řetězec, který je unikátním identifikátorem agendy v Registru práv a povinností. Atribut id však může být ve spojení s base IRI uvedeným v kontextu JSON-LD dokumentu použitý jako absolutní globálně unikátní IRI, které jednoznačně identifikuje agendu napříč všemi datovými sadami. Hodnota atributu id je použita pro odkazování na agendu v rámci otevřených dat Registru práv a povinností a je doporučeno jej používat též pro odkazování na agendu ze všech datových sad veřejné správy ČR.
