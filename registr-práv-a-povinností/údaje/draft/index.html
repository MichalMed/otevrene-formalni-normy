Abstrakt

Tento dokument je otevřenou formální normou ve smyslu § 3 odst. 9 zákona č. 106/1999 Sb., o svobodném přístupu k informacím, pro zveřejňování informací o úkonech z Registru práv a povinností ve smyslu § 51 odst. 1 zákona č. 111/2009 Sb. o základních registrech v podobě otevřených dat. Norma specifikuje způsob publikace na stupni otevřenosti 3* a 5* ve formátu JSON-LD.

JSON struktura

Přehled JSON struktury

Datová sada je tvořena seznamem instancí typu Objekt nebo subjekt údajů uvedených v poli položky.
  • Instance typu Objekt nebo subjekt údajů má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^objekt-subjekt$
    • id : Odkaz dle vzoru ^objekt-subjekt/[0-9]+-[0-9]+$
    • kód : Řetězec dle vzoru ^[0-9]+-[0-9]+$
    • název : Vícejazyčný řetězec
    • popis : Vícejazyčný řetězec
    • ustanovení : seznam instancí typu Ustanovení právního předpisu
    • agenda : Odkaz dle vzoru ^agenda/A[0-9]+$
    • údaje : seznam instancí typu Údaj
  • Instance typu Údaj má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^údaj$
    • id : Odkaz dle vzoru ^údaj/[0-9]+-[0-9]+-[0-9]+$
    • kód-údaje : Řetězec dle vzoru ^[0-9]+-[0-9]+-[0-9]+$
    • název-údaje : Vícejazyčný řetězec
    • popis-údaje : Vícejazyčný řetězec
    • typ-údaje : Řetězec
    • ustanovení-údaje : seznam instancí typu Ustanovení právního předpisu
  • Instance typu Ustanovení právního předpisu má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^ustanovení-právního-předpisu$
    • id : Řetězec dle vzoru ^(.*)$
    • paragraf : Řetězec dle vzoru ^(.*)$
    • odstavec : Řetězec dle vzoru ^(.*)$
    • písmeno : Řetězec dle vzoru ^(.*)$
    • předpis : instance typu Právní předpis
  • Instance typu Právní předpis má následující nepovinné vlastnosti:
    • type : Řetězec dle vzoru ^právní-předpis$
    • id : Řetězec dle vzoru ^(.*)$
    • název-předpisu : Vícejazyčný řetězec
    • číslo-předpisu : Řetězec dle vzoru ^(.*)$
    • rok-předpisu : Řetězec dle vzoru ^(.*)$
