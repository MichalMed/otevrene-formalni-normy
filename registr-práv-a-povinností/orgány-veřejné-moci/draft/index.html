Abstrakt

Tento dokument je otevřenou formální normou ve smyslu , pro zveřejňování informací o orgánech veřejné moci z Registru práv a povinností ve smyslu v podobě otevřených dat. Norma specifikuje způsob publikace na stupni otevřenosti a ve formátu .

Přehled

Datová sada je tvořena seznamem instancí typu Orgán veřejné moci uvedených v poli položky.
  • Orgán veřejné moci obsahuje následující nepovinné vlastnosti:
    • type : IRI třídy orgánů veřejné moci
    • id : IRI orgánu veřejné moci
    • kód : Kód
    • název : Název v různých jazycích
    • vnitřní-organizační-jednotka : Příznak vnitřní organizační jednotky
    • ičo : IČO
    • datové-schránky : Datové schránky (seznam instancí typu Datová schránka)
    • ustanovení : Seznam ustanovení (seznam instancí typu Ustanovení)
    • seznam-kategorií : Zařazení do kategorií (seznam instancí typu Zařazení do kategorie)
    • adresa-místa-pobytu : Adresa místa pobytu z RÚIAN
    • adresa-sídla : Adresa sídla pobytu z RÚIAN
    • osoba-v-čele : Osoby v čele (seznam instancí typu Osoba v čele)
    • zahájení : Datum zahájení výkonu působnosti
    • ukončení : Datum ukončení výkonu působnosti
    • pozastavení-od : Datum pozastavení výkonu působnosti od
    • pozastavení-do : Datum pozastavení výkonu působnosti do
    • přerušení-od : Datum přerušení výkonu působnosti od
    • přerušení-do : Datum přerušení výkonu působnosti do
    • pracoviště-ovm : Seznam pracovišť (seznam instancí typu Pracoviště)
    • nadřízený-ovm : Nadřízený orgán veřejné moci
    • právní-forma : Právní forma
  • Název v různých jazycích obsahuje následující nepovinné vlastnosti:
    • cs : Název v češtině
  • Datová schránka obsahuje následující nepovinné vlastnosti:
    • type : IRI třídy datových schránek
    • id : IRI datové schránky
    • typ-ds : Typ datové schránky
    • identifikátor-ds : Identifikátor
  • Ustanovení obsahuje následující nepovinné vlastnosti:
    • type : IRI třídy ustanovení
    • označení : Označení
  • Zařazení do kategorie obsahuje následující nepovinné vlastnosti:
    • kategorie : Kategorie
    • zařazení-od : Datum zařazení od
    • zařazení-do : Datum zařazení do
  • Osoba v čele obsahuje následující nepovinné vlastnosti:
    • type : IRI třídy osob
    • jméno : Jméno a příjmení
  • Pracoviště obsahuje následující nepovinné vlastnosti:
    • type : IRI třídy pracovišť
    • id : IRI pracoviště
    • adresa-pr : Adresa pracoviště v RÚIAN
    • adresa-pr-txt : Adresa pracoviště textem
    • stát-pr : Stát pracoviště
    • otevření : Datum otevření
    • uzavření : Datum uzavření

Specifikace

V této části jsou definovány jednotlivé typy objektů a jejich vlastnosti. Pro každou vlastnost je uveden její klíč používaný v JSON reprezentaci, její název, datový typ, popis a příklad.
