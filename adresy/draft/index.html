Abstrakt

Tento dokument je otevřenou formální normou ve smyslu § 3 odst. 9 zákona č. 106/1999 Sb., o svobodném přístupu k informacím, pro zveřejňování adres. Je navržena jako sdílená norma pro použití v dalších specifických otevřených formálních normách. Ukazuje, jak reprezentovat informace o oznámeních, vystavených dokumentech, publikujících organizacích a další. Norma je ilustrována příklady dat ve formátech CSV [[rfc4180]], JSON [[ECMA-404]] a XML [[xml]].

Specifikace

V této sekci jsou definovány vlastnosti adresy. Pro každou vlastnost je uveden její identifikátor, který je pro její reprezentaci použit ve všech datových formátech, její název, datový typ, popis a příklad.

Příklady

V této sekci jsou uvedeny příklady reprezentace adresy v různých formátech a v různých úrovních detailu. Datová schémata, která jsou použita v příkladech, lze přímo použít a zahrnout do publikačních plánů, případně rozšířit dle vlastních potřeb. Doporučenými formáty pro publikaci dat z úředních desek jsou JSON a XML, pro která jsou k dispozici příslušná schémata, tedy Schéma pro formát JSON v jazyce JSON Schema a Schéma pro formát XML v jazyce XML Schema. Lze také použít zápis v CSV. Soubor v CSV je nutné doplnit schématem dle specifikace CSV on the Web [[tabular-data-primer]].

Pokud je adresa použita v rámci jiného datového standardu, tak se jména sloupců v CSV tabulce tvoří názvy atributů. Zanořování je řešeno spojením názvu entity z jiného standardu, názvu adresa a z jejího atributu jejich spolením pomocí znaku podtržítka _, například organizace_adresa_psč.

Jednoduchá adresa

Stáhnout JSON 

{
    "číslo_domovní": 12,
    "název_ulice": "Hlavní",
    "název_obce": "Horní Datová",
    "psč": "33101"
}

V XML musí být atributy seřazeny dle specifikace. V případě adresy je to dle abecedy:

Stáhnout XML 

<?xml version="1.0" encoding="UTF-8" ?>
<adresa
    xmlns="https://data.gov.cz/otevřené-formální-normy/adresy/"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="https://data.gov.cz/otevřené-formální-normy/adresy/ 
                        https://data.gov.cz/otevřené-formální-normy/adresy/draft/schémata/adresa.xsd">
    <číslo_domovní>12</číslo_domovní>
    <název_obce>Horní Datová</název_obce>
    <název_ulice>Hlavní</název_ulice>
    <psč>33101</psč>
</adresa>

Stáhnout CSV 

číslo_domovní,název_obce,název_ulice,psč
12,Hlavní,Horní Datová,33101

K souboru CSV je třeba připojit metadata dle standardu CSV on the Web [[tabular-data-primer]]. Například pokud se CSV soubor jmenuje adresa.csv, je s ním publikován i metadatový soubor adresa.csv-metadata.json:

Stáhnout deskriptor 

{
    "@context": [
        "http://www.w3.org/ns/csvw",
        {"@language": "cs"}
    ],
    "url": "1.csv",
    "header": true,
    "tableSchema": {
        "columns": [
            {
                "name": "číslo_domovní",
                "titles": "Číslo domovní",
                "dc:description": "Číslo popisné nebo číslo evidenční, podle rozlišeného typu stavebního objektu."
            }, {
                "name": "název_obce",
                "titles": "Název obce",
                "dc:description": "Název obce."
            }, {
                "name": "název_ulice",
                "titles": "Název ulice",
                "dc:description": "Název ulice, která je navázána na adresní místo. Může být vyplněn pouze u obcí, které mají zavedenu uliční síť."
            }, {
                "name": "psč",
                "titles": "PSČ",
                "dc:description": "Poštovní směrovací číslo"
            }
        ]
    }
}

Jednoduchá adresa s přidanými vlastními atributy

Do schématu si lze v případě potřeby přidávat další atributy. V následujících příkladech jsou nad rámec standardu přidán atribut poznámka.

Stáhnout JSON: 

{
    "číslo_domovní": 12,
    "název_ulice": "Hlavní",
    "název_obce": "Horní Datová",
    "psč": "33101",
    "poznámka": "dole u řeky"
}

V XML se vlastní atributy přidávají na závěr adresy, do vlastního XML jmenného prostoru (XML namespace [[xml-names]]. V tomto příkladu jsou nové atributy přidány do XML jmenného prostoru x.

Stáhnout XML: 

<?xml version="1.0" encoding="UTF-8" ?>
<adresa
    xmlns="https://data.gov.cz/otevřené-formální-normy/adresy/"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="https://data.gov.cz/otevřené-formální-normy/adresy/ 
                        https://data.gov.cz/otevřené-formální-normy/adresy/draft/schémata/adresa.xsd"
    xmlns:x="https://příklad.cz">
    <číslo_domovní>12</číslo_domovní>
    <název_obce>Horní Datová</název_obce>
    <název_ulice>Hlavní</název_ulice>
    <psč>33101</psč>
    <x:poznámka>dole u řeky</x:poznámka>
</adresa>

Alternativně je možné si nadefinovat celé nové XML xchéma, které by vycházelo ze zde uvedeného základního Schématu XML.

Stáhnout CSV 

číslo_domovní,název_ulice,název_obce,psč,poznámka
12,Hlavni,Horní Datová,33101,dole u řeky
