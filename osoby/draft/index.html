Abstrakt

Tento dokument je otevřenou formální normou ve smyslu § 3 odst. 9 zákona č. 106/1999 Sb., o svobodném přístupu k informacím, pro zveřejňování informací o osobách. Je navržena jako sdílená norma pro použití v dalších specifických otevřených formálních normách. Ukazuje, jak reprezentovat identifikaci právnických osob. Norma je ilustrována příklady dat ve formátech CSV [[!rfc4180]], JSON [[!ECMA-404]], XML [[!xml]] a RDF [[!rdf11-concepts]].

Příklady

V této sekci jsou uvedeny příklady reprezentace právnických osob. Datová schémata, která jsou použita v příkladech, lze přímo použít a zahrnout do publikačních plánů, případně rozšířit dle vlastních potřeb. Pro formáty JSON a XML jsou k dispozici příslušná schémata, tedy Schéma pro formát JSON v jazyce JSON Schema [[!json-schema]] a Schéma pro formát XML v jazyce XML Schema [[!xmlschema11-1]]. Lze také použít zápis v CSV. Soubor v CSV je nutné doplnit schématem dle specifikace CSV on the Web [[!tabular-data-primer]].

Pokud je osoba použita v rámci jiného datového standardu, tak se jména sloupců v CSV tabulce tvoří názvy atributů. Zanořování je řešeno spojením názvu entity z jiného standardu, například dodavatel a z názvu atributu osoby jejich spojením pomocí znaku podtržítka _, například dodavatel_ičo.

Osoba

Příklad identifikace tuzemské právnické osoby.

Stáhnout JSON 

{
    "ičo": "00112233",
    "název": "MojeFirma s.r.o."
}

Stáhnout XML 

<?xml version="1.0" encoding="UTF-8"?>
<osoba:osoba xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
xmlns:osoba="https://data.gov.cz/otevřené-formální-normy/osoby/" 
xsi:schemaLocation="https://data.gov.cz/otevřené-formální-normy/osoby/ https://data.gov.cz/otevřené-formální-normy/osoby/draft/schémata/osoba.xsd">
	<osoba:ičo>00112233</osoba:ičo>
	<osoba:název>MojeFirma s.r.o.</osoba:název>
</osoba:osoba>

Stáhnout CSV 

ičo,název
00112233,"MojeFirma s.r.o."

K souboru CSV je potřeba připojit metadatový popisek dle standardu CSV on the Web [[!tabular-data-primer]]:

Stáhnout deskriptor 

{
    "@context": [
        "http://www.w3.org/ns/csvw",
        {"@language": "cs"}
    ],
    "url": "1.csv",
    "header": true,
    "tableSchema": {
        "columns": [
            {
                "name": "ico",
                "titles": "ičo",
                "dc:description": "IČO",
                "datatype": "string"
            }, {
                "name": "nazev",
                "titles": "název",
                "dc:description": "Název právnické osoby.",
                "datatype": "string"
            }
        ]
    }
}

Osoba s přidaným vlastním atributem

Do schématu si lze v případě potřeby přidávat další atributy. V následujících příkladech jsou nad rámec standardu přidán atribut poznámka. Také je ukázán příklad identifikace právnické osoby z jiné země.

Stáhnout JSON: 

{
    "ičo": "00112233",
    "název": "MojeFirma s.r.o.",
    "stát_registrace": {
        "název": "Slovensko",
        "kód": "SVK"
    },
    "poznámka": "Dodavatel"
}

V XML se vlastní atributy přidávají na závěr kontaktu, do vlastního XML jmenného prostoru (XML namespace [[!xml-names]]). V tomto příkladu jsou nové atributy přidány do XML jmenného prostoru x.

Stáhnout XML: 

<?xml version="1.0" encoding="UTF-8"?>
<osoba:osoba xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
xmlns:x="https://příklad.cz" 
xmlns:osoba="https://data.gov.cz/otevřené-formální-normy/osoby/" 
xsi:schemaLocation="https://data.gov.cz/otevřené-formální-normy/osoby/ https://data.gov.cz/otevřené-formální-normy/osoby/draft/schémata/osoba.xsd">
	<osoba:ičo>00112233</osoba:ičo>
	<osoba:název>00112233</osoba:název>
	<osoba:stát_registrace>
		<osoba:název>Slovensko</osoba:název>
		<osoba:kód>SVK</osoba:kód>
	</osoba:stát_registrace>
	<x:poznámka>Dodavatel</x:poznámka>
</osoba:osoba>

Alternativně je možné si nadefinovat celé nové XML schéma, které by vycházelo ze zde uvedeného základního XML schématu.

Stáhnout CSV 

ičo,název,stát_registrace_kód,stát_registrace_název,poznámka
00112233,"MojeFirma s.r.o.",SVK,Slovensko,Dodavatel

                "titles": "název",
                "dc:description": "Název právnické osoby",
                "datatype": "string"
            }, {
                "name": "stat_registrace_nazev",
                "titles": "stát_registrace_název",
                "dc:description": "Název státu registrace",
                "datatype": "string"
            }, {
                "name": "stat_registrace_kod",
                "titles": "stát_registrace_kód",

            }, {
                "name": "poznamka",

                "datatype": "string"
            }
        ]
    }
