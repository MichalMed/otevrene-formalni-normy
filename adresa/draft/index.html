Abstrakt

Tento dokument obsahuje datový standard pro zveřejňování adres jako otevřená data. Je navržen jako sdílený standard pro použití v dalších specifických datových standardech. Ukazuje, jak reprezentovat informace o oznámeních, vystavených dokumentech, publikujících organizacích a další. Je výsledkem diskuze jak s poskytovateli dat o úředních deskách, tak s tvůrci aplikací, které s těmito daty pracují. Standard je ilustrován příklady dat z úředních desek ve formátech CSV [[rfc4180]], JSON [[ECMA-404]] a XML [[xml]].

Specifikace

V této sekci jsou definovány vlastnosti adresy. Pro každou vlastnost je uveden její identifikátor, který je pro její reprezentaci použit ve všech datových formátech, její název, datový typ, popis a příklad.

Příklady

V této sekci jsou uvedeny příklady reprezentace adresy v různých formátech a v různých úrovních detailu. Datová schémata, která jsou použita v příkladech, lze přímo použít a zahrnout do publikačních plánů, případně rozšířit dle vlastních potřeb. Doporučenými formáty pro publikaci dat z úředních desek jsou JSON a XML, pro která jsou k dispozici příslušná schémata, tedy Schéma pro formát JSON v jazyce JSON Schema a Schéma pro formát XML v jazyce XML Schema.

Lze také použít zápis v CSVW. Soubor v CSV je nutné doplnit schématem dle specifikace CSV on the Web [[tabular-data-primer]].

Pokud je adresa použita v rámci jiného datového standardu, tak se jména sloupců v CSV tabulce tvoří názvy atributů. Zanořování je řešeno spojením názvu entity z jiného standardu, názvu `adresa` a z jejího atributu jejich spolením pomocí znaku podtržítka _, například organizace_adresa_psc.

Jednoduchá adresa

V prvním příkladu je ukázán zápis jednoduché adresy.

Zápis v JSON: 

                    {
                        "cislo_domovni": 12,
                        "nazev_ulice": "Hlavní",
                        "nazev_obce": "Horní Datová",
                        "psc": "33101"
                    }

Zápis v XML - zde musí být atributy seřazeny dle specifikace. V tomto případě adresy je to dle abecedy: 

                    <?xml version="1.0" encoding="UTF-8" ?>
                    <adresa>
                        <cislo_domovni>12</cislo_domovni>
                        <nazev_obce>Horní Datová</nazev_obce>
                        <nazev_ulice>Hlavní</nazev_ulice>
                        <psc>33101</psc>
                    </adresa>

Zápis v CSV: 

                    cislo_domovni,nazev_obce,nazev_ulice,psc
                    12,Hlavní,Horní Datová,33101

K souboru CSV je třeba připojit metadata dle standardu CSV on the Web [[tabular-data-primer]]. Například pokud se CSV soubor jmenuje adresa.csv, je s ním publikován i metadatový soubor adresa.csv-metadata.json: 

                    {
                        "@context": [
                            "http://www.w3.org/ns/csvw",
                            {"@language": "cs"}
                        ],
                        "url": "uredni-deska.csv",
                        "header": true,
                        "tableSchema": {
                            "columns": [
                                {
                                    "name": "cislo_domovni",
                                    "titles": "Číslo domovní",
                                    "dc:description": "Číslo popisné nebo číslo evidenční, podle rozlišeného typu stavebního objektu."
                                }, {
                                    "name": "nazev_obce",
                                    "titles": "Název obce",
                                    "dc:description": "Název obce."
                                }, {
                                    "name": "nazev_ulice",
                                    "titles": "Název ulice",
                                    "dc:description": "Název ulice, která je navázána na adresní místo. Může být vyplněn pouze u obcí, které mají zavedenu uliční síť."
                                }, {
                                    "name": "psc",
                                    "titles": "PSČ",
                                    "dc:description": "Poštovní směrovací číslo"
                                }
                            ]
                        }
                    }

Jednoduchá adresa s přidanými vlastními atributy

Do schématu si lze v případě potřeby přidávat další atributy. V následujících příkladech jsou nad rámec standardu přidán atribut poznamka.

Zápis v JSON: 

                    {
                        "cislo_domovni": 12,
                        "nazev_ulice": "Hlavní",
                        "nazev_obce": "Horní Datová",
                        "psc": "33101",
                        "poznamka": "dole u řeky"
                    }

V XML se vlastní atributy přidávají na závěr adresy, do vlastního XML jmenného prostoru (XML namespace [xml-names]. V tomto příkladu jsou nové atributy přidány do XML jmenného prostoru x. 

                    <?xml version="1.0" encoding="UTF-8" ?>
                        <adresa xmlns:x="https://example.com">
                            <cislo_domovni>12</cislo_domovni>
                            <nazev_obce>Horní Datová</nazev_obce>
                            <nazev_ulice>Hlavní</nazev_ulice>
                            <psc>33101</psc>
                            <x:poznamka>dole u řeky</x:poznamka>
                        </adresa>

Zápis v CSV: 

                    cislo_domovni,nazev_ulice,nazev_obce,psc,poznamka
                    12,Hlavni,Horní Datová,33101,dole u řeky

K souboru CSV je opět potřeba připojit metadata dle standardu CSV on the Web [[tabular-data-primer]]: 

                    {
                        "@context": [
                            "http://www.w3.org/ns/csvw",
                            {"@language": "cs"}
                        ],
                        "url": "uredni-deska.csv",
                        "header": true,
                        "tableSchema": {
                            "columns": [
                                {
                                    "name": "cislo_domovni",
                                    "titles": "Číslo domovní",
                                    "dc:description": "Číslo popisné nebo číslo evidenční, podle rozlišeného typu stavebního objektu."
                                }, {
                                    "name": "nazev_obce",
                                    "titles": "Název obce",
                                    "dc:description": "Název obce."
                                }, {
                                    "name": "nazev_ulice",
                                    "titles": "Název ulice",
                                    "dc:description": "Název ulice, která je navázána na adresní místo. Může být vyplněn pouze u obcí, které mají zavedenu uliční síť."
                                }, {
                                    "name": "psc",
                                    "titles": "PSČ",
                                    "dc:description": "Poštovní směrovací číslo"
                                }, {
                                    "name": "poznamka",
